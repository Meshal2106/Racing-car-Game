# Blessed Bites - Restaurant Ordering System

## Description

Blessed Bites is a Restaurant Ordering System built in **Java** using **Swing** for creating a dynamic GUI. It allows users to select from various menu categories, add/remove items from the cart, and place orders with two payment options: **Cash on Delivery** and **Online Payment**. This system also generates an email receipt for customers confirming their order.

## Features

- **Dynamic Menu Display**: View menu categories such as Snacks, Burgers, Pizza, Salads, Cakes, etc., and choose items.
- **Cart Management**: Add or remove items from the cart and update the total price.
- **Order Confirmation**: Customers input their details (Name, Contact, Address) before confirming the order.
- **Payment Options**:
  - **Cash on Delivery**: Enter delivery address details.
  - **Pay Online**: Enter payment source and account details.
- **Receipt Generation**: Display and email order receipt with the itemized order and total price.
